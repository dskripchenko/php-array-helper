## Installation

```
$ php composer.phar require dskripchenko/php-array-helper "@dev"
```

or add

```
"dskripchenko/php-array-helper": "@dev"
```

to the ```require``` section of your `composer.json` file.

* array_merge_deep
* array_sort_deep
* array_is_associative
